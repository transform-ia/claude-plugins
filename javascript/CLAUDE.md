# JavaScript Plugin

Plugin for JavaScript development with ESLint, Prettier, and language server support.

## Structure

- `agents/` - JavaScript dev agent definitions
- `commands/` - Slash commands (cmd-lint, cmd-build, cmd-test)
- `scripts/` - ESLint/build runner scripts
- `skills/` - User-invokable skills

## Commands

- `/javascript:cmd-lint` - Run ESLint
- `/javascript:cmd-build` - Build project
- `/javascript:cmd-test` - Run tests
